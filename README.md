# Wi-Fi Hotspot — Home Assistant add-on repository

Turn the device running Home Assistant OS (e.g. a Raspberry Pi 4) into a
Wi-Fi access point bridged into your LAN — clients join your normal network
as if they had connected to your main Wi-Fi.

## Installation

**Settings → Add-ons → Add-on store → ⋮ → Repositories** → add:

```
https://github.com/domenix/hass-wifi-hotspot
```

Then install **Wi-Fi Hotspot** from the store. Configuration and
troubleshooting live in the add-on's Documentation tab
([DOCS.md](wifi_hotspot/DOCS.md)).
